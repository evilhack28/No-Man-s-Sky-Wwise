# No Man's Sky Wwise Wwise Project

> Reverse engineering the No Man's Sky Wwise Wwise project to recreate the original audio system.

---

## Overview

Reconstruction of the original **No Man's Sky Wwise** Wwise project:

- Audio events
- SoundBanks
- Hierarchies
- RTPCs / States / Switches

---

## Disclaimer

This project will not share any of the original .wav file, it will only replicate the hierarchies and settings of the sound and will never provide any kind or source to this audio file.

## How to Use

- Install **Wwise 2024.1.2** with **Windows** as the deployment platform  
- Download it from [Audiokinetic Download Page](https://www.audiokinetic.com/ja/download)  
- You will need an Audiokinetic account to download the launcher
